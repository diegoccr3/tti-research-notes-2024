Notas de investigación (2024)
Este repositorio contiene borradores técnicos internos desarrollados durante la primera mitad de 2024.
El trabajo explora un marco teórico basado en retrocesos informacionales locales en regímenes de energía extrema, junto con sus posibles firmas observacionales en el CMB.
